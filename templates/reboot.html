Reinicio del Sistema

El dispositivo se reiniciará y se quedará inoperativo durante unos segundos

Una vez reiniciado, el dispositivo cargará la última configuración que haya sido guardada y tendrá que iniciar sesión nuevamente.

Actualización del Sistema

El dispositivo se actualizará y se quedará inoperativo durante unos segundos
Una vez actualizado, el dispositivo cargará la última versión del Software y tendrá que iniciar sesión nuevamente.

Se recomienda no actualizar el dispositivo a no ser que el desarollador se lo indique.
