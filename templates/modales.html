Modo manual

Al activar el modo manual se desactivaran las restricciones y condiciones para poder crear y borrar un evento.
Permitiendo que pueda haber más de un evento simultáneamente y realizar borrado masivos.

Para desactivar el Modo Manual pulse el botón "Modo automático" o cierre e inicie sesión

Modo Automático

Al activar el modo automático se activará las restricciones y condiciones para poder crear y borrar un evento.

Impidiendo que se peuda crear pueda más de un evento simultáneamente y realizar borrado masivos

Los cambios se guardarán automáticamente
